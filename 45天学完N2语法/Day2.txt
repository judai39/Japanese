①～際・～際に・～際は
意味：～のとき。在……时候。「とき」的正式用语。「この際」是「正好趁此机会」的意思。

接続：Ⅴ　辞書形・た形　/　Nの　+　際(に・は)
１．日本からイタリアへ赴任した際、犬を連れていく手続きが大変だった。

２．登録の際に印鑑が必要です。

３．お帰りの際は、忘れ物がないようにご注意ください。

４．ひどい風邪をひいて、のどが痛い。この際、禁煙しよう。



②～に際し/～に際して/～に際しての
意味：～をする前に・～している時に。做……之前；做……的时候。用于特别场合和重要事情的时候。

接続：Ⅴ　辞書形　/　N　+に際し(て)～・に際しての　N
１．山本先生が退職するに際して、何か差し上げたいと思っている。

２．契約に際し、この書類を必ずお読みください。

３．この国では、大統領選挙に際して不正が行われないように監視されている。

４．ご結婚に際してのお祝いを申し上げます。



③～にあたって・～にあたり
意味：～をする前に・～に際して。在……的时候；处于……的情况下。说明为特殊的事情进行的准备，或用在典礼、仪式等的情况下。

接続：Ⅴ　辞書形　/　N　+　にあたって・　にあたり
１．テレビに出演するにあたって、２時間かけてメイクをした。

２．就職するにあたり、スーツを３着買った。

３．開会にあたってのご挨拶を社長の水野から申し上げます。



④～に先立って・～に先立ち・～に先立つ
意味：～の前に。在……之前。表示在开始做某事之前要做的事情。多用于特殊场合，不能用于日常生活。

接続：　Ⅴ　辞書形　/　N　+　に先立って・に先立ち・に先立つ　N
１．試合を始めるに先立ち、両チームの代表が選手宣誓をした。

２．講演を行うに先立って、主催者が挨拶をした。

３．留学に先立つ出費は１００万円以上だった。
