①～次第で・～次第では・～次第だ
意味：～に対応して変わる・～によって決まる。根据......；要看......而定。表示为了适应......，决定什么事情或者结果发生了变化。

接続：　N　+　次第で(は)
１．その日の天気次第で試合をするかどうかが決まります。

２．検査の結果次第では、入院するかもしれません。

３．合格するかどうかは、あなたの努力次第です。

(另外的两个意思分别在day4和day7)



②～によって・～により・～による・～によっては
一、～によって・～により・～による
意味：～に応じてそれぞれ変わる。由于......的不同而......不同。为了对应，说明不同的事物。后面接含有「違う・変わる・いろいろだ」等内容的语句。

接続：　N　+　によって・により・によるN

１．ここから見える景色は季節によって変化する。

２．その日の気分によって、朝食は食べたり食べなかったりする。

３．人により考え方は異なるものだ。

二、～によっては
意味：ある～の場合には。根据......。表示在多种情况下，用于说明在某种情况下，某种场合下......也有可能会......，时使用。

接続：　N　+　によっては

１．仕事が終わる時間によっては、パーティーに間に合わないかもしれない。

２．敬語は言い方によっては相手が嫌な気持ちになることもある。

３．父の体調が悪いので、場合によっては病院へ連れて行くつもりです。

三、～によって・～により・～による
意味：～で。通过......。表示原因和手段、方法等。语气比较生硬。

接続：　N　+　によって・により・によるN

１．パソコンの普及によって、生活が便利になった。

２．話し合いにより、解決した。

３．山火事による死者はとうとう１００人になった。

四、～によって・～により・～による
意味：～に。由......。表示被动句中动作的人物、事物。

接続：　N　+　によって・により・によるN

１．アメリカ大陸はコロンブスによって発見された。

２．この交流会はボランティアグループにより運営されている。



③～によると・～によれば
意味：～の話では。根据......。表示消息的来源。后面经常接续「～そうだ・らしい・ということだ」等语句。

接続：　N　+　によると・によれば

１．天気予報によると、明日は大雨になるらしい。

２．彼女の話によれば、吉田さんは来月結婚するそうだ。

３．資料によれば、あと５０年で世界の石油がなくなるということだ。
