①～にしたがって・にしたがい
意味：～(の変化)と一緒に。随着......。表示随着其他事物的变化而变化。前后语句都可以接续「増える・減る・広がる・～くなる・～になる・～てくる」等表示变化的词语。用于不断变化的事物(不能用于表现只有一次变化的情况)。

接続：Ⅴ　辞書形　/　N　+　にしたがって・にしたがい
１．街の工業化が進むにしたがって、失業率も減ってきた。

２．このグラフから、年齢が高くなるにしたがって、貯蓄が増えていることが分かる。

３．携帯電話の普及にしたがい、家庭電話が必要なくなった。



②～につれて・～につれ
意味：～(の変化)と一緒に。随着......。同「～にしたがって」意思、用法一样。与「～にしたがって」相同，不能用于表示只有一次变化的情况。

接続：Ⅴ　辞書形　/　N　+　につれ(て)
１．台風が近づくにつれて波が高くなるので、海へは行かないでください。

２．ヨーロッパの株価下落につれ、円高になっている。



③～に伴って・～に伴い・～に伴う
意味：～と一緒に。随着......。用于表示随着其他事物变化而发生变化。「～に伴って」也可以用于一次性变化。(例234)

接続：Ⅴ　辞書形　/　N　+　に伴って。に伴い・に伴うN
１．結婚しない人が増加するに伴い、少子化も進んだ。

２．アパートを借りるに伴って、色々な手続きをしなければならない。

３．地震に伴って津波が起こる危険がある。

４．入院に伴う費用は保険で払われるので心配ありません。



④～とともに
一、意味：～(の変化)と一緒に。和......一起。同「～にしたがって」的意思、用法一样。

接続：Ⅴ　辞書形　/　N　+　とともに

１．日本での生活が長くなるとともに、友達が増えてきた。

２．国の経済成長とともに、生活が豊かになった。



二、意味：～と同時に。......的同时。表示两种事情同时发生和两种状态同时出现。

接続：Ⅴ　辞書形　/　イAい　/　ナAである・Nである　+　とともに

１．外国語学習では、単語を覚えるとともに、多くの人と会話することが重要だ。

２．決勝戦に勝って嬉しいとともに、終わってしまって寂しい気持ちもある。

３．吉田さんは、高校の教師であるとともに、大学院の学生でもある。
