①～こそ・～からこそ
意味：「～こそ」是强调，「～からこそ」是强调理由。

接続：　N　+　こそ

　　　　Ⅴ・イA・ナA・N　普通形　+　からこそ

１．「いつもお世話になっております。」

　　「いいえ、こちらこそお世話になっております。」

２．今度こそ合格するつもりだ。

３．困った時こそ助け合うのが本当の友達だ。

４．将来のことを考えるからこそ、今、留学したい。

５．自分の子供がかわいいからこそ叱るんだ。

６．暑い国だらかこそ、辛い物が食べたくなる。



②～さえ・～でさえ
意味：～も・～でも。甚至......连。举出极端的事例，用于说明“......因此其他的事当然也就......”。

接続：N　+　さえ・でさえ

１．昨日は酔っ払って、自分の名前さえ言えなかった。

２．母が病気になった時は水を飲むことさえできなかった。

３．そんなことは子供でさえ知っている。

４．名前さえ書けない。

５．新聞さえ読むひまがない。

６．科学万能の今日でさえ、まだ分からないことはたくさんある。



③～さえ…ば
意味：～だけすれば(あれば)。只要......就。表示要得出后面的结果就必须要这样。

接続：Ⅴ　ます形　+　さえすれば

　　　イAく　/　ナAで　/　N(+で)　+　さえあれば・さえなければ

　　　N　+　さえ　+　Ⅴ　ば形　/　イAければ　/　ナAなら・Nなら

１．この書類を片付けさえすれば、帰れる。

２．体が丈夫でさえあれば、ヨットレース(帆船比赛)に出たい。

３．植物は太陽と水さえあれば生きていける。

４．雨さえ降らなければ、明日はピクニックに行く。

５．交通さえ便利なら、この家で生活するのだが。



④～ないことには
意味：～なければ。没有A就没有B，A是为了B的必须条件的意思。后接否定语句。

接続：　Ⅴ　ない形　/　イAく　/　ナAで・Nで　+　ないことには

１．食事をしないことには体力がつかない。

２．最近の音楽プレーヤーは、小さくて軽くないことには売れない。

３．自分が幸せでないことには、人を幸せすることはできない。

４．実際に読まないことには、この小説の面白さは分からないだろう。
